Option D: Multi-Color Handling

How to handle leaders with 2+ colors

The Problem

Some leaders have 2 colors (e.g., "Red/Blue" starter deck leaders). We need a consistent treatment.

Solutions for Multi-Color

D1

Gradient Border

For 2-color leaders, use a gradient of both colors. Single-color leaders use solid color.

Color Dots

Show small color indicator dots in the header instead of changing the whole card. Scales to any number of colors.

D3

First Color Wins

Use the first color in the array (primary color) for the whole card. Simple, consistent, no special cases.

Comparison
Option Visual Appeal Scalability Effort
D1 — Gradient ★★★ ★★☆ (2 colors max) Low
D2 — Color Dots ★★☆ ★★★ (any count) Low
D3 — First Color ★★☆ ★★★ (any count) Minimal

My take: D3 (First Color) is the pragmatic choice — leaders are 90%+ single-color anyway, and the primary color is what matters for deck building. But D2 (Dots) is the cleanest if you want to show both colors explicitly.
